研究でよく使うファイルやフォルダ（ノーマル）類

フォルダ：
	bi_domain_normal：

Pyファイル：
	cos類似度算出：複数のベクトルに対し，2つで類似度を算出し，総当たりで平均値を出す
	画像トリミング：複数の画像に対し，指定した範囲でトリミングを行う
	画像白黒比率算出：複数の画像に対し，白と黒のピクセル数を算出 比率も算出
	二値化_ガウシアン_膨張収縮：ガウス適応的二値化法で二値化し，膨張収縮を行う
	没_適応的大津法：適応的二値化法に大津法を組み込んだが没

IPYNBファイル：
	一括ノイズフィルター：複数の画像に対し，ノイズフィルターを行う
	画像最大エントロピー算出：1枚の画像に対し，閾値を変えながらエントロピーを算出
	MRF：ノイズ除去
